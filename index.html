BEFORE YOU INSTALL

What you'll need.

We're mobile-first, not desktop-required. Most of the course lives in your pocket — but to do the full 90 days you'll want occasional access to a computer.

Your phone

The main classroom. The full 90-day path is do-able entirely in the in-app chat with Claude built in. You've already got this. ✓

A computer (optional pairing)

Pair the app with claude.ai on a laptop to unlock the deeper web experience — Projects, Artifacts, Skills & Connectors — and the Claude Code CLI for Days 57–90 (builder lessons). The mobile lessons teach the same concepts via copy-paste prompts.

No computer right now? You can still progress every day on mobile — the in-app chat covers all the concepts. Just come back to the hands-on steps when a laptop is handy. Each lesson shows a small badge so you'll always know which is which.
